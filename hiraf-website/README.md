# موقع وكالة حِرف الرقمية (Hiraf)

موقع تعريفي فاخر بصفحة واحدة (One-Page) لوكالة تسويق رقمي سعودية، مبني
بأحدث معايير التصميم الحديث مع تجربة عربية كاملة (RTL).

## التقنيات

- **Next.js 14** (App Router) — أداء و SEO.
- **Tailwind CSS** — نظام تصميم وألوان مخصص.
- **Framer Motion** — حركات عند التمرير (Fade / Slide / Scale) و Parallax.
- **Lucide React** — الأيقونات.
- خطوط عربية فاخرة: **IBM Plex Sans Arabic** و **Tajawal** و **Cairo**.

## التشغيل

```bash
npm install
npm run dev      # وضع التطوير على http://localhost:3000
npm run build    # بناء الإنتاج
npm start        # تشغيل الإنتاج
```

## الأقسام

شريط تنقل (Sticky/Blur) · القسم الرئيسي (Hero + Parallax + Particles) ·
الفلسفة · الخدمات · باقات الشعارات · باقات الهوية البصرية (مع ضمان حِرف) ·
التصاميم المختارة · الإحصائيات (Count-up) · تأسيس المتاجر الإلكترونية ·
إدارة وسائل التواصل · الإعلانات الممولة · آراء العملاء (Carousel) ·
لماذا حِرف؟ · التواصل (نموذج) · التذييل.

## المزايا

- تصميم متجاوب بالكامل (جوال / تابلت / دسكتوب).
- وضع ليلي (Dark Mode) قابل للتبديل مع حفظ التفضيل.
- حركات عند التمرير، Glassmorphism، Gradient Text، شريط متحرك (Marquee).
- جميع المحتوى والأسعار بالريال السعودي (ر.س) كما هو مطلوب بالكامل.

## المحتوى

كل النصوص والباقات والأسعار موجودة في `lib/data.ts` لتسهيل التعديل.
