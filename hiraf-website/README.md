# موقع حِرف | Hiraf Conversion Site

A premium, high-conversion single-page website for **Hiraf** (حِرف) — a digital agency/studio. Built for one goal: turn visitors into WhatsApp inquiries and customers.

## Highlights

- **Decision-first hero** — visitors choose their need before seeing services (`أين تريد أن يبدأ مشروعك؟`).
- **4 interactive choice cards** (متجر إلكتروني · هوية بصرية · مبيعات أكثر · إدارة حسابات) that expand inline to reveal relevant packages — no page navigation.
- **Premium pricing packages** with `⭐ الأكثر طلباً` highlight, price, duration, full content, and a WhatsApp request button.
- **Animated trust counters**, **portfolio showcase**, **glassmorphism RTL auto-sliding reviews**, and a large **conversion CTA**.
- **Floating WhatsApp button** (always visible) and every CTA routes to WhatsApp.
- Fully **Arabic RTL**, responsive, dark-green brand theme, zero build step.

## Usage

Open `index.html` in any browser, or deploy the folder to any static host (Netlify, Vercel, GitHub Pages, etc.). Everything is self-contained in a single file (HTML + CSS + JS), only the Tajawal Google Font is loaded externally.

## Customizing packages & prices

Package names, prices, durations, and contents are placeholder-ready inside `index.html` under each `<div class="panel" id="panel-...">`. Edit the text directly to match your real offers.
